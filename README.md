## 📖 Sobre o Projeto
Este projeto é uma página estática desenvolvida em **HTML e CSS** que apresenta informações sobre um destino turístico.  
O objetivo é praticar conceitos de front-end e criar uma página visualmente agradável para exibir conteúdo turístico.

### 🎥 Demonstração
👉 [Visualizar Projeto no GitHub Pages](https://dodogm.github.io/Local-Turistico/)  



### 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- Git & GitHub
